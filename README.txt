CARTA DE CUMPLEAÑOS - Abuela Adriana
=====================================

Archivos del proyecto:
- carta-cumpleanos.html  → Abrir este archivo en el navegador
- styles.css            → Estilos
- app.js                → Funciones (sonido, melodía, lightbox, etc.)
- imagenes/             → Fotos (imagen1.jpeg, etc.)

Cómo usar:
1. Abrir carta-cumpleanos.html en Chrome, Edge o Firefox
2. Hacer clic en "Abrir" para ver la carta
3. Clic en las fotos para verlas en grande
4. Doble clic en cualquier lado para más confeti
5. "Ver de nuevo" vuelve al sobre cerrado

Para compartir: enviar toda la carpeta o subir a un servidor web.
